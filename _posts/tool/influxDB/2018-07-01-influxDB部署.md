# 部署


```shell
# 下载

cd ~/software

wget https://repos.influxdata.com/rhel/6Server/x86_64/stable/influxdb-1.2.0.x86_64.rpm

sudo rpm -ivh influxdb-1.2.0.x86_64.rpm

sudo vi /etc/influxdb/influxdb.conf


[admin]
  enabled = true
  bind-address = ":8083"

[http]
  enabled = true
  bind-address = ":8086"


```


# 启动

```shell
service influxdb start
```
