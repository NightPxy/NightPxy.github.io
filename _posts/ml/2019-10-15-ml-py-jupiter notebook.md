# 前置  

基于操作系统,下载OpenSSL包  

```
# Linux 直接安装即可 
# Windows 如下
https://slproweb.com/products/Win32OpenSSL.html
```



# 安装  

Anaconda 环境下  

```shell

# 安装
conda install jupyter notebook
# 启动
jupyter notebook
```

# 配置  

```shell
jupyter notebook  --generate-config

# c.NotebookApp.notebook_dir 工作台目录
```
